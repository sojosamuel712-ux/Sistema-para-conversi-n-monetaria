Este sistema fue diseñado con el fin de diseñar una plataforma open-source, tipo conversor
de moneda local a internacional, creando asi este diseño en principal.En si esta tiene mas 
como proposito para uso personal, ya que ademas de ejercer las respectivas conversiones lleva
un apartado de control de gastos y ahorro que desee guardar y registrar, de esa forma podra
llevar un control facil y rapido de sus finanzas.

Se ejecutaran las respectivas modificaciones dependiendo del uso que se le de, cambiando codigo
predeterminado como la moneda, el tipo de conversion e incluso la api o sistema que utilizara
para que el sistema detecte el valor de una su moneda local a precio internacional.
